## Installing

```sh
$ npm install --save ion-universal-nav
```

## Quickstart

#### Import

Import **ion-universal-nav** module in Angular app.

```typescript
import { IonUniversalNavModule } from 'ion-universal-nav'

(...)

@NgModule({
  (...)
  imports: [
    IonUniversalNavModule.forRoot()
  ]
  (...)
})
```
